## Project Description
The Online Event Registration System (OERS) enables users to:

- Browse available events
- View event details
- Register for events
- Receive confirmation emails
- Make online payments

Administrators can:

- Add, edit, and delete events
- View registration reports
- Manage event details

This repository contains documentation links only, as required for the assignment.
A fully working source code is not mandatory for this Software Project Planning project.
